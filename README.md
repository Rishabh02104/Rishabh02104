## ⚡ Core Focus & Featured Projects

### 🤖 [AI Job Agent](https://github.com/Rishabh02104/AI_Job_Agent)
* **What**: An autonomous, agentic job-hunting pipeline.
* **Stack**: FastAPI, LangChain, Next.js, Supabase, Playwright.
* **Highlight**: Uses LLM orchestrators to scrape job boards, score resume match accuracy, and automate browser-driven application submissions.

### 🧠 [RishavendraOS](https://github.com/Rishabh02104/RishavendraOS)
* **What**: An interactive, WebGL-powered 3D portfolio operating system.
* **Stack**: Next.js, React Three Fiber (R3F), Three.js, Framer Motion, Custom GLSL Shaders.
* **Highlight**: Features a custom-rendered 3D holographic brain navigation system, volumetric lighting, and matrix rain watermark depth-masking.

### 📄 [CareerForge AI](https://github.com/Rishabh02104/Careerforge-ai)
* **What**: An ATS optimization platform and AI interview coach.
* **Stack**: Next.js, TypeScript, Tailwind CSS, OpenAI API, Supabase.
* **Highlight**: Parses resumes against job profiles, structures feedback loops, and acts as a dynamic mock-interview voice simulator.

### 🛸 [Drone Binary Terrain Mapping](https://github.com/Rishabh02104/drone-binary-terrain-mapping)
* **What**: CNN-based terrain patch classification and estimation.
* **Stack**: Python, TensorFlow/PyTorch, OpenCV.
* **Highlight**: Processes aerial mapping feeds to segment terrain features and compute road dimension metrics dynamically.

### 🔐 [Secure Voting](https://github.com/Rishabh02104/secure-voting)
* **What**: A visual cryptography-based voting system concept.
* **Stack**: JavaScript, HTML5 Canvas.
* **Highlight**: Reconstructs vote shares mechanically using split overlay transparency grids, protecting ballot secrecy without digital keys.
